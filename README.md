# React Activity Calendar

A React component to display activity data in a calendar (heatmap).

**[Demo (Storybook)](https://grubersjoe.github.io/react-activity-calendar)**

![Screenshot](screenshot.png)

## Development

### Start the Storybook

```shell
yarn storybook
```

### Update docs

```shell
yarn build:storybook
```
